PDF File (Dynamic e-Chapter 7) - Neural Networks
Attached is the latest version of e-Chapter 7 (pdf file). Please do not redistribute in any format. The e-Chapter is offered for free without warranty of any kind.

Our goal with the e-Chapters is to cover new topics and update existing topics as the trends in Machine Learning change. Hence, the e-Chapters are intended to be dynamic and build upon the core foundation laid out in the book.

Please use this subforum to post any comments or questions about this e-Chapter, including typos. After the typos are corrected in a new version of the pdf file, the errata threads will be archived.

If you would like to cite this e-chapter, please use:

Yaser Abu-Mostafa, Malik Magdon-Ismail and Hsuan-Tien Lin. Learning From Data, e-Chapter 7: "Neural networks," Amlbook.com, March, 2012.
